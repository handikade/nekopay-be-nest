# User List Implementation

Implement a paginated, searchable, and sortable list of users, restricted to administrative access.

## Objectives
- Implement `UserService.findAll` with pagination, filtering (search), and sorting.
- Add `GET /users` endpoint to `UserController`.
- Ensure strict access control (Admin only).
- Validate input and output using Zod schemas.

## Technical Details

### 1. Schema and DTO Review
- [x] Verify `UserQueryParamsSchema` in `src/user/user.schema.ts` supports `page`, `limit`, `search`, `sortBy`, and `sortOrder`.
- [x] Verify `UserListSchema` and `UserPresentationSchema` in `src/user/user.schema.ts` correctly omit sensitive data like `password`.
